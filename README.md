# Government Intent Engine (GIE) – Demo

واجهة تجريبية لمحرك تحليل نية المستخدم (Intent Engine) موجه لخدمات منصة أبشر. يهدف المشروع إلى اقتراح الخدمات الحكومية المناسبة بناءً على النص الذي يدخله المستخدم، مع واجهة تحاكي تجربة تطبيق الجوال.

## محتويات المشروع

GIE_project/
├─ backend/

│  └─ gie_engine.py
├─ frontend/
│  ├─ index.html

## متطلبات التشغيل

- Python 3.9 أو أحدث  
- متصفح حديث (Chrome / Edge)  
- Visual Studio Code  
- إضافة Live Server في VS Code  

## تشغيل المشروع محليًا

### تشغيل السيرفر الخلفي (Backend)

افتحي الطرفية داخل مجلد backend ثم شغّلي:

python app.py

سيعمل السيرفر على:
http://127.0.0.1:5000

للتحقق من حالة السيرفر:
http://127.0.0.1:5000/api/health

يجب أن تظهر النتيجة:
{ "status": "ok" }

### تشغيل الواجهة الأمامية (Frontend)

1. افتحي ملف frontend/index.html في VS Code  
2. اضغطي بزر الفأرة الأيمن  
3. اختاري Open with Live Server  

سيتم فتح الواجهة على:
http://127.0.0.1:5500/frontend/index.html

## كيفية الاستخدام

- أدخلي نصًا عربيًا واضحًا مثل:
  - أبغى أسافر
  - أحتاج حجز موعد
  - سداد الرسوم الحكومية
- اضغطي زر "بحث" أو Enter
- سيقوم النظام بتحليل النية واقتراح الخدمات المناسبة
- ستظهر الخدمات المفضلة والخدمات المقترحة في بطاقات تفاعلية
- يمكن الضغط على أي خدمة للانتقال إليها (محاكاة)

## التحقق من صحة الإدخال

- في حال إدخال نص غير متعلق بالخدمات الحكومية أو نص عشوائي، تظهر رسالة:
الرجاء إدخال نص صحيح وواضح.
- يتم فلترة النص قبل إرساله للسيرفر لضمان تجربة مستخدم أفضل

## التقنيات المستخدمة

- Frontend: HTML, CSS, JavaScript  
- Backend: Python (Flask)  
- Communication: REST API باستخدام JSON  
- UI Design: واجهة تحاكي تطبيق الجوال  

## الغرض من المشروع

- عرض فكرة Intent-Based Navigation  
- تحسين تجربة المستخدم في الوصول للخدمات الحكومية  
- تقديم نموذج أولي (Prototype) قابل للتطوير مستقبلاً
